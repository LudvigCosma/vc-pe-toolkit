## What's in Here?

A growing collection of AI workflows, skills, and playbooks for PE and VC professionals. Things that help you actually get work done — screening decks, tearing down CIMs, reviewing contracts, running DD on a market using books and research, monitoring portfolio KPIs.

Everything is open-source and platform-agnostic. Use what works for the job.

### What's a Skill?

Portable instructions that teach an AI agent how to perform a specific task your way. Drop a `SKILL.md` into Claude, Cowork, or any agent that supports custom instructions — it handles the rest.

### What's a Workflow?

A step-by-step playbook for getting a specific job done with AI tools. Could involve Gemini for processing long documents, Cowork for automating file tasks, Claude for analysis, or a combination. Less "plug and play" than a skill, but often more powerful for complex tasks.

## Get Started

### Skills

Each skill folder contains a `SKILL.md` file. To use one:

1. Navigate to the skill you need
2. Copy the contents of `SKILL.md`
3. Add it to your AI agent as a custom skill or system prompt
4. Follow the skill's `README.md` for any setup steps

### Workflows

Each workflow folder contains a `WORKFLOW.md` or `README.md` explaining the process:

1. Read through the workflow steps
2. Set up any tools mentioned (Gemini, Cowork, Claude, etc.)
3. Follow the playbook — adapt it to your stack and process

> [!TIP]
> Skills and workflows are platform-flexible. Something built for Claude will often work in other agents with minor tweaks. Workflows that reference Gemini can usually be adapted to any LLM with a long context window.

## Contributing

We welcome contributions from anyone working in PE, VC, growth equity, or adjacent fields.

### What You Can Submit

| Type | Description | Example |
|------|-------------|---------|
| **Skill** | A `SKILL.md` that an AI agent can follow to complete a specific task | CIM teardown prompt for Claude |
| **Workflow** | A step-by-step playbook for using AI tools to accomplish a goal | Using Gemini to process industry books during DD |
| **Template** | Reusable output formats, report structures, or analysis frameworks | IC memo template with AI-generated sections |

### Quick Submit (no Git required)

**[Submit here!](https://github.com/LudvigCosma/vc-pe-toolkit/issues/new?template=submit-resource.yml)** — Fill out the form and we'll handle the rest.

### Submit via Pull Request

1. Fork this repository
2. Create a folder under the appropriate directory with a descriptive name
3. Include:
   - `SKILL.md` or `WORKFLOW.md` — the core instructions or playbook
   - `README.md` — what it does, when to use it, which tools it requires
   - Sample files or templates if applicable
4. Open a pull request with a short summary of the use case

### Quality Guidelines

- **Focused** — One resource, one job. A skill that extracts key terms from an LPA is better than one that tries to do all of legal review.
- **Specific** — Be clear about inputs, outputs, and which tools are needed.
- **Tested** — If it works reliably in your workflow, it's ready.
- **Clean** — No proprietary data. Use synthetic or anonymized examples only.

> [!NOTE]
> All submissions are reviewed for quality and security before being added.

See [CONTRIBUTING.md](CONTRIBUTING.md) for the full guide.

## Why Open Source This?

Most PE/VC teams are building AI workflows behind closed doors, which makes sense for proprietary strategy — but not for operational tooling. The mechanics of parsing a CIM or formatting an LP report are not competitive advantages. Sharing them makes everyone faster.
