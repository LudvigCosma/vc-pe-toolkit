# Pitch Deck Triage

Structured pitch deck analysis for VC/PE funds.

**Not a scoring system. A question-surfacing system.**

## Quick Start

1. Upload a pitch deck (PDF or PPTX) to Claude
2. Paste the contents of `SKILL.md` or reference it in a Claude Project
3. Get IC-ready summary + full analysis

## Output

- **IC Summary** — 15-second Slack-ready triage
- **Deep Dive** — Full analysis for founder calls

## Customization

Add `investment-thesis.md` (see `/templates`) for thesis-fit scoring.
